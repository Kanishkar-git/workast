# Privacy Policy — Workast Bot (Zoho Cliq)

This Privacy Policy explains how the Workast Bot (“we”, “our”, “the bot”) handles data inside Zoho Cliq.

## 1. Information We Collect
- **User ID & Name:** Required to identify the user in Zoho Cliq.
- **Workast API Token:** Stored securely only to authenticate requests to Workast.
- **Team/Space IDs:** Used only to fetch spaces, tasks, and notes.
- **Channel Mapping Details:** If the user maps a channel for notifications.

We do NOT collect emails, phone numbers, passwords, or personal content outside what is required for bot functionality.

## 2. How We Use Data
- To connect your Workast account to Zoho Cliq.
- To fetch tasks, notes, spaces, or notifications from Workast.
- To send alerts inside channels mapped by the user.
- To perform task operations as authorized by the user.

## 3. Data Sharing
We DO NOT sell, rent, or share any user data with third parties.

Data is only exchanged between:
- Zoho Cliq
- Workast API  
And only when the user triggers a command.

## 4. Data Storage
- API tokens are stored securely inside Zoho Cliq’s database.
- Tokens are encrypted and accessible only to the bot.

## 5. Data Retention
Users may unlink their account at any time, which removes all stored data.

## 6. Changes
We may update this Privacy Policy at any time. Updates will be reflected in this document.
